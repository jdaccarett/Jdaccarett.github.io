April 2017

Glucose Guardian

The applications sole purpose is to provide a functional calculation that will yield exact amounts of the fast acting sugar source (skittles) to bring your levels back to normal.Example if you have a low blood sugar reading of 60mg/dl the formula will require your target level(the level you want to elevate your glucose level to) and your current blood glucose value. It also required your insulin sensitivity (how sensitive your body is to the effects of insulin) which is given by your endocrinologist, as well as your insulin to carb ratio (how many units of insulin you need per carb). Next the application runs the calculation and it will give you the exact amount of skittles to eat (i.e. i would need 17 skittles to raise my glucose level from 60mg/dL to 120mg/dL (+-10mg/dL)) .

Technologies Used

  • Node.js - Express - HTML -CSS - Javascript - Twillo API - Google Maps - Mustache

Real Estate leads

A node.js project that enables Real Estate agents to gather leads in their social media platforms. Running on a node.js server that connects to the Zillow Zestimate API. This web application will output the estimated value of a current home. The user inputted information will be collected into a MySQL database and displayed to be used for sale leads..

  • Node.js/Express
  • HTML/CSS
  • Javascript
  • MySQL

Exercise Log

Visiting the page will show a table displaying all completed exercises. At the top of the page, there is be a form that lets you enter in all the data needed to make a new exercise entry in the table. Hitting the submit will add the exercise to the table if it was successfully added to the database. Hitting the delete button will immediately remove the exercise from the table and from the database. Hitting the edit button should make it possible to edit your exercise.

  • Node.js/Express

  • Javascript
  • MySQL
